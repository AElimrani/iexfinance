IEX Market Data¶

The following functions retrieve data from the IEX Market Data endpoints.

  • TOPS
  • Last
  • DEEP
  • Book

Warning

IEX Market Data endpoints may return empty or raise an exception outside of market hours.

TOPS¶

TOPS is IEX’s aggregated best quoted bid and offer position in near real time.

Access is available through the top-level function get_market_tops():

iexfinance.get_market_tops(symbols=None, output_format='json', **kwargs)¶

Top-level function to obtain TOPS data for a symbol or list of symbols

Parameters:
  • symbols (str or list, default None, optional) – A symbol or list of symbols
  • output_format (str, default 'json', optional) – Desired output format.
  • kwargs – Additional Request Parameters (see base class)

Usage¶

In [1]: from iexfinance import get_market_tops

In [2]: get_market_tops('AAPL')
Out[2]: 
[{'symbol': 'AAPL',
  'sector': 'technologyhardwareequipment',
  'securityType': 'commonstock',
  'bidPrice': 0,
  'bidSize': 0,
  'askPrice': 0,
  'askSize': 0,
  'lastUpdated': 1536955200321,
  'lastSalePrice': 223.7,
  'lastSaleSize': 100,
  'lastSaleTime': 1536955199794,
  'volume': 760575,
  'marketPercent': 0.02413}]

Note

The /tops endpoint without any parameters will return all symbols. TOPS data with all symbols is 1.78mb uncompressed (270kb compressed) and is throttled at one request per second, per IEX docs

Last¶

Last is IEX real-time trade data from the IEX book. This endpoint allows retrieval of a real-time quote.

Access is available through the top-level function get_market_last():

iexfinance.get_market_last(symbols=None, output_format='json', **kwargs)¶

Top-level function to obtain Last data for a symbol or list of symbols

Parameters:
  • symbols (str or list, default None, optional) – A symbol or list of symbols
  • output_format (str, default 'json', optional) – Desired output format.
  • kwargs – Additional Request Parameters (see base class)

Usage¶

In [3]: from iexfinance import get_market_last

In [4]: import pandas as pd

In [5]: df = get_market_last(symbols="AAPL", output_format='pandas')

In [6]: df['price']
Out[6]: 
0    223.7
Name: price, dtype: float64

Note

The /tops/last endpoint without any parameters will return all symbols.

DEEP¶

DEEP is IEX’s aggregated real-time depth of book quotes. DEEP also provides last trade price and size information.

Access is available through the top-level function get_market_deep():

Note

Per IEX, DEEP only accepts one symbol at this time.
