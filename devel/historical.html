Historical Data¶

Historical time series data is available through the top-level get_historical_data method, which sources the chart endpoint.

iexfinance.get_historical_data(symbols=None, start=None, end=None, output_format='json', **kwargs)¶

Top-level function to obtain historical date for a symbol or list of symbols. Return an instance of HistoricalReader

Parameters:
  • symbols (str or list, default None) – A symbol or list of symbols
  • start (datetime.datetime, default None) – Beginning of desired date range
  • end (datetime.datetime, default None) – End of required date range
  • output_format (str, default 'json', optional) – Desired output format (json or pandas)
  • kwargs – Additional Request Parameters (see base class)
Returns:

Historical stock prices over date range, start to end
Return type:

list or DataFrame

Data can be retrieved from up to 5 years before the current date.

Usage¶

If no date parameters are passed, the start date will default to 2015/1/1 and the end date will default to the current date.

In [1]: from iexfinance import get_historical_data

In [2]: from datetime import datetime

In [3]: start = datetime(2017, 2, 9)

In [4]: end = datetime(2017, 5, 24)

In [5]: f = get_historical_data('AAPL', start, end, output_format='pandas')

In [6]: f.loc["2017-02-09"]
Out[6]: 
open      1.286506e+02
high      1.294275e+02
low       1.281327e+02
close     1.294031e+02
volume    2.834986e+07
Name: 2017-02-09, dtype: float64

Plotting¶

With Pandas output formatting, we are able to plot historical price movements using matplotlib.

In [7]: from iexfinance import get_historical_data

In [8]: from datetime import datetime

In [9]: import matplotlib.pyplot as plt

In [10]: start = datetime(2017, 2, 9)

In [11]: end = datetime(2017, 5, 24)

In [12]: f = get_historical_data("AAPL", start, end, output_format='pandas')

In [13]: plt.plot(f["close"])
Out[13]: [<matplotlib.lines.Line2D at 0x2b30c03d9e80>]

In [14]: plt.title('Time series chart for AAPL')
Out[14]: Text(0.5,1,'Time series chart for AAPL')

In [15]: plt.show()
